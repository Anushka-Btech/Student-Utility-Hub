## Student Utility Hub

Built a mini full-stack Student Utility System focused on solving everyday student needs in a structured way.

The system integrates multiple functionalities including authentication, expense tracking, and task management within a single workflow.

Worked on designing a clean flow between frontend, backend, and data handling to ensure a smooth and logical user experience.

This project reflects my focus on moving beyond isolated features toward building connected and scalable systems.

Continuing to learn by building and improving real-world applications.
